Basic HTML structure

HTML is organized, kind of like a family tree or a smaller box within a larger box. When an element (i.e. smaller box) is inside another element (i.e. larger box) it's considered the child of that element. The child element (i.e. smaller box) is said to be nested inside of the parent element (i.e. bigger box). Attributes are what goes inside those individual elements; they allow stuff to happen or changes to be made. Values are what goes inside attributes to specify what happens or what changes are being made.

A variety of tags, attributes and elements

  • The HTML Head element <head> contains machine-readable information (metadata) about the document, like its title, scripts, and style sheets.
  • The HTML Title element <title> defines the document's title that is shown in a browser's title bar or a page's tab.
  • The HTML Body element <body> represents the content of an HTML document - basically the stuff that you see when you open a webpage.
  • href <a>, <area>, <base>, <link> - The URL of a linked resource.
  • <img> used with the attributes scr="" and alt="" can embed images.
  • id - Often used with CSS to style a specific element.
  • class - Often used with CSS to style elements with common properties.
